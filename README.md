# App UNO - Taskflow Project
Aplicación web sencilla para gestionar tareas:  
[App UNO][1] cómo práctica de DAM en [Corner-Estudios][2]
 [1]: https://taskflow-project-virid.vercel.app
 [2]: https://www.corner-estudios.com/

 - Herramientas: **HTML + CSS + JavaScript** y se despliega en **Vercel**.

# Estructura del proyecto

├── index.html - Estructura principal de la app

├── styles.css - Estilos y maquetación (estilo: Efecto 2000)

└── app.js - Lógica de gestión de tareas y localStorage

**Resultados de las pruebas de la aplicación (13/3/2026) :**
 - Todo funcionando correctamente a falta de la migración completa a Tailwind y añadir más funcionalidades.
 - Falta crear foco en el panel de cada tarea para poder controlarlas desde el teclado.

**Mejoras Pendientes:**
 - Añadir aria-label cuando se perfeccione el diseño responsive.
 - Hacer que al clicar una categoría de la barra lateral filtre las tareas.
 - Sustituir el "Eliminar tarea" por un **Archivo** del que poder recuperar las tareas y visualizar estadisticas en el tiempo (anuales, mensuales, últimos X días.)
 - El panel de nueva tarea ocupa mucho, hacer que el recuadro dónde se escrive se adapte al tamaño de la pantalla pero permita que si escribes mucho las letras se salgan del recuadro.
